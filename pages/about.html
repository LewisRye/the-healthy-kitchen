Statement of Originality

This is a statement of originality for the coursework submission for the 'Web 3D Applications' module. The work presented here is my own and has not been submitted for any other assessment.

All 3D models, animations, and sound effects used in this project are either created by me or sourced from free resources with proper attribution.

About

This website promotes healthy food choices by simulating an interactive 3D kitchen. It showcases detailed models of nutritious foods with educational insights and visual appeal.

3D Models

Fridge

The fridge opens and closes interactively, revealing foods inside. Animations and sound effects enhance realism. There is also glass shelves inside the fridge that are transparent, making it a more detailed mesh with realistic properties.

Banana

A textured 3D model of a banana rotates including hover tooltips and a custom painted texture in Blender. The texture of the banana has been drawn to include slight brown spots, enhancing realism involving the ripeness of the banana.

Cherry

The cherry model demonstrates simple clustering and rotation effects using jQuery.

Apple

Apple clusters can be zoomed and spun using mouse interaction, giving a tactile experience.

Design & Integration

Upon loading the site, the user enters a kitchen with interactive objects. The user can look around and interact with objects and go to different scenes. Bootstrap ensures responsiveness across devices, such as mobile, and consistent UI appearance/behaviour.

Interaction & Implementation

Interactive elements were tested thoroughly, including clickable animations and UI feedback. Examples of interactivity include the fridge, which opens and closes, the light switch, which can be toggled by the user, and the cupboards which take the user to a chopping board scene where they can cut open a banana.

Going Beyond

Models

The models I have created in Blender are of sufficient complexity, with correct roughness and metallic properties. I have even created the textures myself using the 'Texture Paint' tool.

GitHub

This project is version-controlled using Git, with full commit history available on the GitHub repository linked in the navbar.

Bootstrap

All layout and UI elements utilize Bootstrap 5.3, ensuring a consistent dark theme and accessibility.

jQuery

jQuery handles dynamic interactivity such as showing and hiding buttons during interactions, an example of this is the 'Close Fridge' button that only shows when the fridge is open.
